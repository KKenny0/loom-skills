# Loom Draft

Write article prose from a trusted research package.

## Read First

- `../shared/references/writing-pipeline.md`
- `../shared/references/companion-skills.md`
- `../shared/references/synthesis-pack-schema.md`
- `../shared/references/schema.md`

## Inputs

- Prefer a Synthesis Pack. Accept reviewed Daily Notes only when the user explicitly wants a lighter workflow.

## Outputs

- Draft: content-complete version for user feedback.
- Final: publish-ready version when requested.
- Default destination: `01_Daily_Notes/YYYY/working/`.

## Companion Skills

- Use `ljg-writes` for essay drafting.
- Use `baoyu-format-markdown` for Markdown cleanup after content is settled.
- Use `ljg-plain` only to clarify hard passages, not to flatten the user's voice.

## Rules

- Draft/Final may polish, restructure, and strengthen personal voice.
- Draft/Final must not overwrite or contaminate Raw Capture, Daily Note, Source Brief, Synthesis Pack, or Topic Note.
- Do not move content to `published/` unless the user explicitly confirms publication readiness.
- Preserve high-risk claim markers until verified or consciously accepted.

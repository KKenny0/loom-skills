# Loom Read

Turn one source into a Source Brief.

## Read First

- `../shared/references/writing-pipeline.md`
- `../shared/references/companion-skills.md`
- `../shared/references/source-brief-schema.md`

## Inputs

- Raw Capture, inbox Daily Note, URL/file already captured, or one Material List row.

## Output

- Source Brief following `../shared/references/source-brief-schema.md`.

## Companion Skills

- Use `ljg-read` for article/text close reading.
- Use `ljg-paper` for papers.
- Use `ljg-paper-river` when the user asks for paper lineage or intellectual background.
- Use `ljg-plain`, `ljg-learn`, or `ljg-think` for explanation and concept decomposition when useful.

## Rules

- Separate source claims, evidence, limits, disputes, quotable lines, and claims to verify.
- Preserve the source's meaning and uncertainty.
- Do not merge claims across sources; that belongs to `loom-synthesis-pack`.
- Do not write Draft/Final or Topic Notes in this stage.

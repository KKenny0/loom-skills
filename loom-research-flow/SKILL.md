# Loom Research Flow

Run the research-only pipeline.

## Pipeline

1. `loom-intake`
2. `loom-capture`
3. `loom-read`
4. `loom-synthesis-pack`

## Read First

- `../shared/references/writing-pipeline.md`
- `../shared/references/companion-skills.md`
- Stage schemas as needed.

## Inputs

- URLs, topic keywords, files, folder, pasted text, or mixed materials.

## Outputs

- Material List.
- Raw Captures and inbox Daily Notes.
- Source Briefs.
- Synthesis Pack.

## Stop Point

Stop after the Synthesis Pack. Do not write Draft/Final unless the user asks for `loom-article-flow` or `loom-draft`.

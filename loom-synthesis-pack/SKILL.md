# Loom Synthesis Pack

Create the research package that precedes article writing.

## Read First

- `../shared/references/writing-pipeline.md`
- `../shared/references/companion-skills.md`
- `../shared/references/synthesis-pack-schema.md`

## Inputs

- Material List plus Source Briefs, multiple Daily Notes, or multiple captured sources.

## Output

- Synthesis Pack following `../shared/references/synthesis-pack-schema.md`.

## Companion Skills

- Use `ljg-rank` to reduce themes to core generators.
- Use `ljg-think` for root-cause or first-principles analysis.
- Use `ljg-roundtable` for conflict, critique, and multi-perspective synthesis.
- Use `ljg-learn` for concept anatomy when needed.

## Rules

- Separate merged conclusions, conflicts, consensus/disagreement, user thesis, high-risk claims, and evidence weights.
- Include Source Coverage Checklist.
- Do not collapse unresolved disputes into a false consensus.
- Do not draft publication prose except as outline or thesis notes.
